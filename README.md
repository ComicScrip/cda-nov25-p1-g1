# Gettting Started backend

## Install dependencies

```
cd backend
npm i
```

## Setup environment variables

```
cp .env.example .env
```

And then adjust variables in ```.env``` to match your local environment.

## Generate example data

```
npm run resetDB
```

## Start the server in dev mode

```
npm run dev
```
